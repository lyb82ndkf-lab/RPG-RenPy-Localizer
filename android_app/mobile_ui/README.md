# Android Mobile UI

这里是 Android 壳内加载的移动工作台界面，目前采用单文件 `index.html`，方便直接塞进 WebView 和快速迭代。

## 当前能力

- 固定 1360 x 820 工作台比例，并自动按手机横屏缩放。
- 游戏目录选择、项目载入、RPG Maker MV/MZ 启动、Ren'Py 启动、外部 exe 运行器入口。
- 翻译、数据、地图、运行五个工作区的移动端骨架。
- 触屏按键布局编辑器：
  - 可以拖动按钮位置。
  - 可以修改按钮文字、映射键位、大小、透明度。
  - 提供 RPG 和 Ren'Py 两套预设。
  - 保存到浏览器 `localStorage`。
  - 在 Android 壳内会通过 `RPGRenPyShell.saveTouchControls()` 同步到原生覆盖层。

## 设计方向

移动端按“横屏掌机控制台”设计，而不是简单压缩桌面 UI。核心原则是：

- 手机横屏优先。
- 触控按钮必须足够大，并允许玩家自己调整。
- 游戏画面和工具界面之间可以快速切换。
- 长任务仍交给后端/原生层，避免 WebView 页面卡死。

## 后续计划

- 把实时作弊控制移植成移动端卡片。
- 增加摇杆组件，不只支持方向键按钮。
- 增加布局导入/导出，方便分享按键配置。
- 接入 Android 原生悬浮工具面板，而不是只在 WebView 内切换。
