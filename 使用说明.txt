RPGRenPyLocalizer 使用说明

1. 运行 RPGRenPyLocalizer.exe。
2. 点击“选择 exe”，选择游戏的启动 exe。
3. 点击“载入”，工具会识别 RPG Maker MV/MZ 或 Ren'Py 游戏。
4. Ren'Py 游戏如果需要深度提取，工具会自动安装提取脚本并提示选择已打开的游戏窗口；没有已打开窗口时会自动启动一次游戏。
5. 文本提取完成后，可以在“翻译工作台”里筛选文件、全选当前筛选、使用 AI 或百度 API 翻译。
6. AI/API Key 会保存到当前 Windows 用户配置目录，不会放进打包目录。
7. 分享给别人时，请整体打包 dist\RPGRenPyLocalizer 文件夹，不要只发单独 exe。

提示：如果某些 Ren'Py 游戏有启动器或加载动画窗口，建议先手动打开到真正游戏窗口，再回到工具选择该窗口进行深度提取。
