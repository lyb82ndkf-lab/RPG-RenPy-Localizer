发布给朋友时请打包整个 dist\RPGRenPyLocalizer 文件夹。

必须包含：
- RPGRenPyLocalizer.exe
- _internal 文件夹

朋友电脑不需要安装 Python。
这个发布版由 PyInstaller 打包，_internal 里已经包含 Python 运行时和工具依赖。

不要只发送单独的 RPGRenPyLocalizer.exe。
如果缺少 _internal 文件夹，程序会无法启动或缺少运行库。

AI 配置和 API Key 默认保存在用户自己的 C 盘配置目录：
%APPDATA%\RPGRenPyLocalizer\settings.json

因此你打包 dist\RPGRenPyLocalizer 发给别人时，不会带上你自己的 API Key。
